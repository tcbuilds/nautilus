# Example Project Chamber

This is a sanitized example of the three artifacts Nautilus expects a project to produce before implementation work gets serious:

- `mvp.md` — product shape and constraints.
- `CLAUDE.md` — project-specific AI collaboration rules.
- `implementation_plan.md` — phased, checkable roadmap.

The example is intentionally small. It shows artifact shape without binding the playbook to a real customer, service, repository, or deployment.
